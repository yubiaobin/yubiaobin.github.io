git create（git 创建命令）

Clone an existing repository（克隆远程已存在的版本库）

{% highlight ruby %} $ git clone ssh://user@domail.com/repo.git {% endhighlight %}

{% highlight ruby %} $ git clone https://user@github.com/someone/repo.git {% endhighlight %}

Create an new local repository（创建本地的本版库）

{% highlight ruby %} $ git init {% endhighlight %}

local changes（本地的更改命令）

Changed files in your working directory（本地目录更改状态）

{% highlight ruby %} $ git status {% endhighlight %}

Changes to tracked files（本地目录追踪文件更改的比较）

{% highlight ruby %} $ git diff {% endhighlight %}

Add all current changes to the next commit（添加当前所有的更改去暂存区以备提交）

{% highlight ruby %} $ git add {% endhighlight %}

Add some changes in to the next commit（添加当前file文件所有的更改去暂存区以备提交）

{% highlight ruby %} $ git add -p {% endhighlight %}

Add all changes in all range to the next commit（添加当前目录所有文件所有的更改去暂存区以备提交）

{% highlight ruby %} $ git add --all {% endhighlight %}

Commit previously staged changes（提交之前暂存区的更改）

{% highlight ruby %} $ git commit {% endhighlight %}

Commit all local changes in tracked files（提交之前所有本地追踪文件的更改）

{% highlight ruby %} $ git commit -a {% endhighlight %}

Change the last commit（查看上次提交的更改的部分）

{% highlight ruby %} $ git commit --amend {% endhighlight %}

Add a message at this commit（添加这次提交更改的文字说明）

{% highlight ruby %} $ git commit -m {% endhighlight %}
