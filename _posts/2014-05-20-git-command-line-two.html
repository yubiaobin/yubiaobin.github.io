Commit history（提交历史）

Show all commits , staring with newest（从最近的提交开始显示所有的提交）

{% highlight java %} $ git log {% endhighlight %}

Show changes over time for a specific file（特定文件的按时间显示提交历史）

{% highlight java %} $ git log -p {% endhighlight %}

Who changed what and when in （什么时候谁更改了文件）

{% highlight java %} $ git blame {% endhighlight %}

Branches & Tags（分支和标识）

List all existing branches（列出所有现有的分支）

{% highlight java %} $ git branch {% endhighlight %}

Switch HEAD branch（切换到这个分支）

{% highlight java %} $ git checkout {% endhighlight %}

Create a new branch based on your current HEAD（在当前版本库创建一个新的分支）

{% highlight java %} $ git branch {% endhighlight %}

Create a new tracking branch based on a remote branch（在远程版本库创建一个新的跟踪的分支）

{% highlight java %} $ git branch --tracked {% endhighlight %}

Delete a local branch（删除一个本地的分支）

{% highlight java %} $ git branch -d {% endhighlight %}

Mark the current commit with a tag（标记最新的提交）

{% highlight java %} $ git tag {% endhighlight %}

Publish your local all tags to remote （发布本地所有的标记去远程本版库）

{% highlight java %} $ git push --tags {% endhighlight %}
