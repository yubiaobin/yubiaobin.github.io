ES6一些新语法

let与const 关键字

可以把let看成var，只是它定义的变量被限定在了特定范围内才能使用，而离开这个范围则无效。const则很直观，用来定义常量，即无法被更改值的变量。

1，let可以看做是块级作用域内定义变量的关键词。

以前我们这样写js；

{% highlight javascript %} var a = 5; if(true){ var a = 10; } console.log(a); //10 {% endhighlight %}

现在我们用ES6语法来写js；

{% highlight javascript %} let a = 5; if(true){ let a = 10; } console.log(a); //5 {% endhighlight %}

let作用在块级作用域中，所以不管是switch还是if还是for，只要是let定义的变量，他就只能在那个花括号内部起作用。

2，const用来定义常量。

一旦定义一个变量为const类型，后面就不能对他进行修改。

{% highlight javascript %} const aa = 100; console.log(aa); //100 aa = 200; console.log(aa); //100 {% endhighlight %}

const 代表一个值的 常量索引 （事实上绝大多数语言都提供这样的功能）。换句话说，变量名字在内存中的指针不能够改变，但是指向这个变量的值 可能 改变。

内置Array.prototype.find()和内置Array.prototype.filter();

1，Array.prototype.find()查找到第一个符合条件的值。

{% highlight javascript %} var arr1 = [1,2,3,4,5]; var arr2 = arr1.find(function(key){return key > 2}); console.log(arr2); // [3] {% endhighlight %}

2，Array.prototype.filter()查找到所有符合条件的值。

{% highlight javascript %} var arr1 = [1,2,3,4,5]; var arr2 = arr1.find(function(key){return key > 2}); console.log(arr2); // [3,4,5] {% endhighlight %}
