Git create（git 创建命令）

Clone an existing repository（克隆远程已存在的版本库）

{% highlight java %} $ git clone ssh://user@domail.com/repo.git {% endhighlight %}

{% highlight java %} $ git clone https://user@github.com/someone/repo.git {% endhighlight %}

Create an new local repository（创建本地的本版库）

{% highlight java %} $ git init //（当前目录初始化——即增加.git隐藏文件来记录版本） {% endhighlight %}

Local changes（本地的更改命令）

{% highlight java %} $ git status //（本地目录状态） {% endhighlight %}

{% highlight java %} $ git diff //（本地目录追踪文件更改的比较） {% endhighlight %}

{% highlight java %} $ git add --all //（添加当前所有有过改动的文件） {% endhighlight %}

{% highlight java %} $ git commit //（提交更改的记录到暂存区） {% endhighlight %}

{% highlight java %} $ git commit -a //提交之前所有本地追踪文件的更改） {% endhighlight %}

{% highlight java %} $ git commit --amend //查看上次提交的更改的部分） {% endhighlight %}

{% highlight java %} $ git commit -m //（添加这次提交更改的文字说明） {% endhighlight %}

Commit history（提交历史）

{% highlight java %} $ git log //（从最近的提交开始显示所有的提交） {% endhighlight %}

{% highlight java %} $ git reflog //（从最近的提交开始显示所有的提交） {% endhighlight %}

{% highlight java %} $ git log -p //（特定文件的按时间显示提交历史） {% endhighlight %}

{% highlight java %} $ git log --pretty=oneline //（单行显示所有的提交） {% endhighlight %}

Who changed what and when in （什么时候谁更改了文件）

{% highlight java %} $ git blame {% endhighlight %}

Branches & Tags（分支和标识）

List all existing branches（列出所有现有的分支）

{% highlight java %} $ git branch {% endhighlight %}

Switch HEAD branch（切换到这个分支）

{% highlight java %} $ git checkout {% endhighlight %}

Create a new branch based on your current HEAD（在当前版本库创建一个新的分支）

{% highlight java %} $ git branch {% endhighlight %}

Create a new tracking branch based on a remote branch（在远程版本库创建一个新的跟踪的分支）

{% highlight java %} $ git branch --tracked {% endhighlight %}

Delete a local branch（删除一个本地的分支）

{% highlight java %} $ git branch -d {% endhighlight %}

Mark the current commit with a tag（标记最新的提交）

{% highlight java %} $ git tag {% endhighlight %}

Publish your local all tags to remote （发布本地所有的标记去远程本版库）

{% highlight java %} $ git push --tags {% endhighlight %}

Update & Publish（更新和发布）

List all currently configured remotes（列出所有当前的远程配置）

{% highlight java %} $ git remote -v {% endhighlight %}

Show information about a remote（显示远程版本库信息）

{% highlight java %} $ git remote show {% endhighlight %}

Add mew remote repository , named （增加一个新的名为的远程版本）

{% highlight java %} $ git remote add {% endhighlight %}

Download all changes from , but don't integrate into HEAD（从远程下载所有更改，但是不和本地合并）

{% highlight java %} $ git fetch {% endhighlight %}

Download changes and directly merge/integrate into HEAD （从远程下载所有更改，并且和本地合并）

{% highlight java %} $ git pull {% endhighlight %}

Publish local changes on a remote（发布本地的提交的更改去远程版本库）

{% highlight java %} $ git push {% endhighlight %}

Delete a branch on the remote（删除远程的版本库的一个分支）

{% highlight java %} $ git push : {% endhighlight %}

Publish your local all tags to remote （发布本地所有的标记去远程本版库）

{% highlight java %} $ git push --tags {% endhighlight %}

Merge & Rebase（合并，重定义）

Merge into your current HEAD（合并分支去当前版本）

{% highlight java %} $ git merge {% endhighlight %}

Rebase your current HEAD onto （重定义当前的版本）

{% highlight java %} $ git rebase {% endhighlight %}

Abort a rebase（中支重定义）

{% highlight java %} $ git rebase --abort {% endhighlight %}

Continue a rebase after resolving conflicts（解决冲突之后继续）

{% highlight java %} $ git rebase --continue {% endhighlight %}

Changes to tracked files（本地目录追踪文件更改的比较）

{% highlight java %} $ git diff {% endhighlight %}
