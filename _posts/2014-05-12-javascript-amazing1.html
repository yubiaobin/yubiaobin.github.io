Javascript的链式作用域

对于JavaScript新手来说scoping是最令人困惑的部分之一。事实上，不仅仅是新手，我遇到或很多有经验的JavaScript程序员也不能完全理解scoping。JavaScript的scoping如此复杂的原因是它看上去非常像C系语言的成员。请看下面的C程序：

函数作用域问题

{% highlight html %} var a = 1; function saya(){ a = 10; return; function a(){ alert("I'm another a") } } saya(); alert(a) // 1 {% endhighlight %}

{% highlight html %} var a = 1; function saya(){ a = 10; return; alert("I'm another A") } saya(); alert(a) // 10 {% endhighlight %}

{% highlight html %} var a = 1; function saya(){ a = 10; return; function abc(){ alert("I‘m abc") } } saya(); alert(a) // 10 {% endhighlight %}
