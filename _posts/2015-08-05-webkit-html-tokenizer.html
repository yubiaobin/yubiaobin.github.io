HTML Tokenizer

HTML词法解析器

立即执行的函数表达式（IIFE）是一种常见的创建自己作用域范围的代码方式。

{% highlight javascript %} (function(){ // the function expression is surrounded by parenthesis // variables defined here // can't be accessed outside })(); // the function is immediately invoked {% endhighlight %}

一般我们写，习惯在之前添加一个分号；已便于（保证）JS语法解释器合理的解析我们写的代码，而不会抛出异常。

{% highlight javascript %} ;(function(){ // the function expression is surrounded by parenthesis // variables defined here // can't be accessed outside })(); // the function is immediately invoked {% endhighlight %}
