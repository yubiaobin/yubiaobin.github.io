鼠标的四种状态

鼠标的状态样式代码

按照正常的顺序，应该这样写样式代码

{% highlight css %} a:link {color:#000;} a:visited {color:#000;} a:hover {color:#000;} a:active {color:#000;} {% endhighlight %}

当然你也可以这样写

{% highlight css %} a:link {color:#000;} a:active {color:#000;} a:visited {color:#000;} a:hover {color:#000;} {% endhighlight %}

但是一般规则link定义在最前；hover在link之后；
