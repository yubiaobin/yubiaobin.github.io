WebApp页面布局

1.meta标签

{% highlight html %} {% endhighlight %}

第一个标签的意思：强制文档的宽度等于（移动）设备的宽度，初始话缩放比例为1，最大缩放比例为1，且不允许用户点击屏幕放大浏览
第二个标签的意思：此定义是iphone手机中safari私有属性，允许全屏浏览
第三个标签的意思：也是iphone私有标签，表示safari顶端状态条的样式是黑色
第四个标签的意思：告诉设备忽略页面中的数字，不将其识别为电话号码。

2.html5标签

尽量选择语义化的html5标签多使用，比如使用article标签定义一个文章或段落，使用section定义一个内容块，使用nav标签定义一个导航。

3.利用CSS3属性

  • CSS3 可以写圆角属性；
  • CSS3 可以写背景渐变属性；
  • CSS3 可以写阴影属性；

4.学会使用webkit-box

手机端的自适应布局 依靠webkit内核为display属性提供了一个webkit-box的值，它可以帮助前端工程师做到盒子模型灵活控制。
