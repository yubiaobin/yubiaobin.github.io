git create（git 创建命令）

Clone an existing repository（克隆远程已存在的版本库）

{% highlight ruby %} $ git clone ssh://user@domail.com/repo.git {% endhighlight %}

{% highlight ruby %} $ git clone https://user@github.com/someone/repo.git {% endhighlight %}

Create an new local repository（创建本地的本版库）

{% highlight ruby %} $ git init {% endhighlight %}

local changes（本地的更改命令）

Changed files in your working directory（本地目录更改状态）

{% highlight ruby %} $ git status {% endhighlight %}

Changes to tracked files（本地目录追踪文件更改的比较）

{% highlight ruby %} $ git diff {% endhighlight %}

Add all current changes to the next commit（添加当前所有的更改去暂存区以备提交）

{% highlight ruby %} $ git add {% endhighlight %}
