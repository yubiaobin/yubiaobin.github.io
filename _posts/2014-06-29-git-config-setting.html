git config(用户设置命令)

git 设置全局用户名

{% highlight html %} $ git config --global user.name {% endhighlight %}

{% highlight html %} Example: $ git config --global user.name "yubiaobin" {% endhighlight %}

git 设置全局用户邮箱

{% highlight html %} Example: $ git config --global user.email yubiaobin@163.com {% endhighlight %}
